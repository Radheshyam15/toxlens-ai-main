⬡ Technology

How ToxLens Works

⬡
01

SMILES Input

Accepts molecular SMILES strings and validates chemical structure integrity via RDKit processing.

→
🔬
02

Graph Conversion

Molecules are converted to graphs — atoms as nodes, bonds as edges — with extracted Morgan Fingerprints.

→
🧠
03

GNN Inference

PyTorch Geometric GNN processes graph data, generating toxicity probability via sigmoid activation.

→
⚡
04

Risk Assessment

Combined toxicity outputs generate LOW / MEDIUM / HIGH interaction risk with overall confidence score.

API Integration

🔑 Authentication

All API endpoints (except /health) require an X-API-Key header. Contact us to obtain a production key. Demo keys are available for testing.

🛡️ Rate Limiting

Each API key is limited to 10 requests per minute. Exceeding this returns HTTP 429 Too Many Requests. Contact us for enterprise-tier rate limits.
